# Website Yayasan Sunan Ampel v1
Sistem informasi profil yayasan, pendaftaran santri, dan berita kegiatan.

## Fitur Utama
- Profil Yayasan & Video Profil
- Dashboard Admin (Filament)
- Manajemen Berita & Galeri
- Kontak WhatsApp Admin

## Cara Instalasi Lokal
1. `git clone`
2. `composer install`
3. `cp .env.example .env` (Lalu sesuaikan database)
4. `php artisan key:generate`
5. `php artisan storage:link`
